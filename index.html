Sobre Mí

¡Hola!, Soy un apasionado del mundo de la programación y el diseño web. Aunque recién estoy empezando mi camino en el desarrollo de software, me fascina la idea de aprender y dominar diferentes lenguajes de programación para crear páginas web únicas y con un enfoque innovador.

Proyectos realizados

Blog Personal Informativo

Descripción: Un blog sencillo donde compartes artículos sobre tus intereses. Incluye imágenes, texto y enlaces. Características: Diseño responsivo que se adapta a diferentes tamaños de pantalla. Navegación sencilla con un menú en la parte superior. Secciones de artículos con imágenes y texto. Tecnologías: HTML para la estructura, CSS para el diseño y JavaScript para la interactividad básica (como un botón de “leer más”).

Galería de Imágenes

Descripción: Una página web que muestra una colección de imágenes organizadas en una cuadrícula. Características: Diseño responsivo con imágenes que se ajustan al tamaño de la pantalla. Efectos de hover para resaltar las imágenes. Modal para ver las imágenes en tamaño completo. Tecnologías: HTML para la estructura, CSS para el diseño y JavaScript para la funcionalidad del modal.

Página de Contacto Descripción:

Una página de contacto donde los visitantes pueden enviarte mensajes. Características: Formulario de contacto con campos para nombre, correo electrónico y mensaje. Validación de formularios para asegurar que los campos estén completos antes de enviar. Diseño responsivo que se adapta a dispositivos móviles y de escritorio. Tecnologías: HTML para la estructura del formulario, CSS para el diseño y JavaScript para la validación del formulario.
